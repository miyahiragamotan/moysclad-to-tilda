В коне создаём файл .env
Заполняем его сделующими переменными
token_sclad = API Токен пользователя, можно получить в найстройках после авторизации на сайте
filter_name = По какому полю фильтруем товары для выгрузки, в случае дополнительных полей это URL (meta) этого поля, пример: https://api.moysklad.ru/api/remap/1.2/entity/product/metadata/attributes/000bb00-0d00-00f0-0a00-00e000000e0a
filter_value = Значение поля для фильтрации
id_sale_price = ID цены для выгрузки, пример: 0000000-0000-0000-0000-00e00000000a
