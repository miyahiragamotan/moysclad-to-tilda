В коне создаём файл .env \n
Заполняем его сделующими переменными \n
token_sclad = API Токен пользователя, можно получить в найстройках после авторизации на сайте \n
filter_name = По какому полю фильтруем товары для выгрузки, в случае дополнительных полей это URL (meta) этого поля, пример: https://api.moysklad.ru/api/remap/1.2/entity/product/metadata/attributes/000bb00-0d00-00f0-0a00-00e000000e0a \n
filter_value = Значение поля для фильтрации \n
id_sale_price = ID цены для выгрузки, пример: 0000000-0000-0000-0000-00e00000000a \n
file_path_csv = export_of_goods.csv \n
file_path_yml = export_of_goods.yml \n
default_company = Название компании \n
default_category = Название категории по умолчанию \n
default_brand = Название бренда по умолчанию \n
default_quantity = 999 / Остаток по умолчанию \n
